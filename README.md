# TrancheReady — AML/CTF Tranche 2 Pack Generator

Upload **Clients** and **Transactions** CSVs → get:
- Ranked **risk scores** with explainable notes
- **Monitoring cases** (large cash, structuring, high-risk corridors)
- A sector-aware **Program** (HTML)
- An **Evidence Pack (ZIP)** with JSON and a **SHA-256 manifest**
- A tokenized **share link** with an online **verification** page

## Run locally
```bash
npm install
node server.js
# open http://localhost:3000
